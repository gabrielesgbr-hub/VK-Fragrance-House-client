# VK-Fragrance-House-client

# Definición del objetivo
Sitio web con el objetivo de implementar un comercio en línea con información de la empresa y pasarelas de pago para la empresa de venta de perfumes “VK Fragrance House” con el objetivo de automatizar la el registro y la realización de las ventas de sus productos, brindando una plataforma con opciones de compra y de contacto para las clientes y una forma fácil para el administrador de agregar nuevos productos y llevar un registro de sus ventas para así poder tomar decisiones acertadas en el futuro. “VK Fragrance House” es un emprendimiento real que opera en la ciudad de Mérida, Yucatán, es una empresa pequeña y por el momento cuenta con un catálogo de 21 perfumes y envíos solo a la misma ciudad, pero el contar con una plataforma web centralizada el negocio puede escalar a uno más grande.

# Identificación de los usuarios
El público objetivo de la aplicación son personas interesadas en adquirir perfumes en la ciudad de Mérida, Yucatán a un costo accesible y con un vendedor de confianza.
